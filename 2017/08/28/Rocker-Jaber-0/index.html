Rocker && Jabber

Aug 28 2017

近一个月没有更博了，忙啥呢 ？

比较官方的回答：忙项目／忙升职加薪／忙娶媳妇儿。

实话讲：忙着给 Rocker && Jabber 铲屎 =,=

很多成年后的需求都可以追溯到儿时的愿望

我小时候养过啥？好像除了蚕宝宝就没有了。要么在家里不样养，要么在学校没条件。

工作两年，基本解决了生存问题，衣食住行 OK，各种安全保障无忧。根据马斯洛需求层次理论，下一个目标其实就是归属问题，所谓情感归属的需要。

友情爱情 ？Nope, 真不缺(要的也不多)；缺什么？细细想来，应该是缺少一种，被 TA 需要的感觉。

生孩子？ Not ready！

那么，小时候曾经想养小猫而不能的事儿，恰逢其时。

Rocker: 这么可爱亲人的蓝孩纸注定要好吃好喝好住一辈子

下定决心之后，日思夜想。终于有天在微博上找到了一个流浪猫舍。

一个简易的猫舍，大大小小几十只猫子，两个阿姨轮流照顾(纯个人自发救助，点赞)。大致看了一下照片，相中了一只白肚子的狸花猫。

到了猫舍，第一眼就认出来 Rocker，意外的是，Rocker 看到我们，也立马跑过来蹭蹭蹭了。

哈哈，有种多年不见，父子相认的感觉。

PS: Rocker 从领养的时候就一直鼻腔呼吸道有炎症，声音沙哑，所以得名 Rocker。回来不久就一直在医院打针吃药，希望能够早点完全康复。

Jabber: 你这样的菇凉还是稍微减个肥吧

Jabber 是一只在娘胎里就被我们预定了的英国短毛猫。

在朋友那里见到她第一反应，胖。果然是吃妈妈奶 + 各种营养品长大的啊哈哈哈。

第二个反应：脸确实大。为什么要加一个 “确实” 呢？因为她还在哺乳期的时候，他爸爸的照片会时不时出现，那脸。。真的是。。。只能感慨基因很强大 Orz

因为爸爸叫杰米，所以给她取个中文名：杰宝（杰米的宝贝），傲娇的小公举，一言不合就喜欢哈人。

不上 JB 你说个图 ？？？
